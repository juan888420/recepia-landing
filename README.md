# Recepia

**AI receptionist on WhatsApp for appointment-based businesses.**

![Next.js](https://img.shields.io/badge/Next.js-000000?logo=nextdotjs&logoColor=white)
![TypeScript](https://img.shields.io/badge/TypeScript-3178C6?logo=typescript&logoColor=white)
![Supabase](https://img.shields.io/badge/Supabase-3FCF8E?logo=supabase&logoColor=white)
![Claude API](https://img.shields.io/badge/Claude_API-D97757?logo=anthropic&logoColor=white)

Recepia answers WhatsApp messages, books and reschedules appointments, and sends reminders automatically, so clinics and service businesses never miss a client because nobody picked up the phone. Built first for dental clinics, designed to scale to any appointment-based vertical.

This repo hosts the marketing site. The product itself is closed-source.

## Features

- Conversational booking: clients schedule, reschedule or cancel directly in WhatsApp, in natural language
- Calendar-aware: checks real availability before offering time slots
- Automatic reminders that reduce no-shows
- Handles FAQs (prices, hours, location) without human intervention
- Dashboard for the business owner: appointments, conversations and metrics in one place

## Tech Stack

- **Frontend:** Next.js, TypeScript, Tailwind CSS
- **Backend:** Supabase (Postgres, Auth, Realtime)
- **AI:** Claude API for conversation handling
- **Automation:** n8n workflows, WhatsApp Business API
